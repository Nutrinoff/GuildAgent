Installation

Follow these steps to install and set up the Guild Agent bot:
  1. Use the /setup command to set up the 3 necessary channels.
  2. Use the /setuptimers to set up the age a post must be before it's reposted, and how many days old a post should be before it's removed.
  3. Use /setupvalues to add these values:
  4. Google sheet ID (it must be public!)
  5. The Column Range of the sheet (i.e A1:T which is the standard in the provided Form and Sheet).
  6. The Timestamp header name (for exclusion in the message posts), it's just "Timestamp" if you use the provided form and sheet.
  7. The Guild Logo header name (for fetching the image without including URL in the message), it's just "Guild Logo" if you use the provided form and sheet.

If you encounter any issues, please refer to our troubleshooting guide or contact support.
